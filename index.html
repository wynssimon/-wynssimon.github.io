Mijn naam is Simon Wyns, 20 jaar uit Antwerpen, en ik studeer Grafische & Gigitale Media aan de AP Hogeschool te Antwerpen. Ik verwacht af te studeren in juni 2023. Tijdens mijn opleiding heb ik geproefd van alle vakgebieden zoals branding, publication, motion en aldus ook interfaces. Daardoor heb ik nu een stevige basis van al deze zaken. Sinds januari 2022 heb ik besloten me volledig te richten op interface en sinds september enkel nog op coding. Hier wil ik later ook mee aan de slag gaan in de bedrijfswereld, maar eerst ben ik van plan om nog een graduaat Programmeren te volgen zodat ik het totaalpakket front-end en back-end beheers. Momenteel beheers ik qua coding de talen: HTML, CSS, JavaScript en PHP. Van React JS en React Native heb ik de basis meegekregen maar besef ik dat ik nog zeer veel moet bijleren.
